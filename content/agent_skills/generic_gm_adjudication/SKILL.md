# Generic GM Adjudication

Use this capability to decide whether a player input is a question, proposal, action, passive
choice, unsupported authority claim, or memory recall.

Do not assume any concrete game system. If a risky and uncertain action requires resolution, request
the currently loaded ruleset extension. If the content package adds special GM requirements, use
those requirements only after they are retrieved for the current session.
